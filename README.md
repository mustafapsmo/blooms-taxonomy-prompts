# Bloom’s Taxonomy: Prompts for Generating Questions
This repository contains a structured reference of **Bloom’s Taxonomy verbs and prompts** to support educators, instructional designers, and curriculum developers in creating effective learning objectives and assessment questions.

## Overview
Bloom’s Taxonomy categorizes cognitive skills into six hierarchical levels:

1. Remembering
2. Understanding
3. Applying
4. Analyzing
5. Evaluating
6. Creating

Each level includes action verbs that help frame questions and learning outcomes.
